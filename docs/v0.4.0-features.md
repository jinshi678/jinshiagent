**代码示例：**

```python
from jinshiagent.core import Agent
from jinshiagent.llm import LLMClient, LLMConfig

agent = Agent(
    name="my-agent",

```

### 4. 多 Agent 协作 ✅（v0.3.0）

| 模式 | 实现类 | 说明 |
|------|--------|------|
| 主从模式 | `Orchestrator` | 主 Agent 拆解任务 → 分配子 Agent → 汇总结果 |
| 流水线模式 | `Pipeline` | 多阶段顺序执行，前一阶段输出作为下一阶段输入 |
| 团队模式 | `AgentTeam` | 统一管理多个 Agent，支持三种协作方式动态切换 |

**共同特性：**
- 支持 LLM 驱动模式（智能调度）和规则模式（无 LLM）
- 同步 `run()` + 异步 `arun()` 双模式

**代码示例：**

```python
from jinshiagent.core.multi_agent import Orchestrator, Pipeline, AgentTeam, TeamMode, SubAgent

# Orchestrator 主从模式
orch = Orchestrator(
    master_agent=coordinator,
    sub_agents=[
        SubAgent(name="weather", agent=weather_agent, description="查询天气"),
        SubAgent(name="writer", agent=writer_agent, description="撰写建议"),
    ],
)
result = orch.run("帮我查北京天气并写一份出行建议")

# Pipeline 流水线模式
pipeline = Pipeline(agents=[weather_agent, writer_agent])
result = pipeline.run("帮我查北京天气并写一份出行建议")

# AgentTeam 团队模式
team = AgentTeam(name="travel", mode=TeamMode.ORCHESTRATOR, master=coordinator)
team.add_member(weather_agent)
team.add_member(writer_agent)
result = team.run("帮我查北京天气并写一份出行建议")
```

### 5. MCP 协议支持 ✅（v0.4.0 新增）

| 组件 | 实现类 | 功能 |
|------|--------|------|
| MCP 客户端 | `MCPClient` | 连接、握手、工具发现、工具调用 |
| Stdio 传输层 | `StdioTransport` | 基于 asyncio 子进程的 stdio 通信 |
| HTTP 传输层 | `HTTPTransport` | 基于 HTTP POST 的远程通信 |
| 工具适配器 | `MCPToolAdapter` | 将 MCP 工具注册到 ToolRegistry |
| 配置模型 | `MCPServerConfig` | Pydantic 配置类 |
| 工具信息模型 | `MCPToolInfo` | 工具描述数据结构 |
| 模拟服务器 | `mock_server` | 本地测试用 MCP 服务器 |

**MCP 使用流程：**

```python
import asyncio
from jinshiagent.mcp import MCPClient, MCPServerConfig, MCPToolAdapter
from jinshiagent.core import Agent, ToolRegistry

async def main():
    # 1. 配置
    config = MCPServerConfig(
        name="my-tools",
        transport="stdio",  # 或 "http"
        command="python",
        args=["server.py"],
    )

    # 2. 连接 + 发现工具
    client = MCPClient(config)
    tools = await client.connect()
    print(f"发现工具: {list(tools.keys())}")

    # 3. 适配到 ToolRegistry
    registry = ToolRegistry()
    adapter = MCPToolAdapter(client, registry, prefix="mcp_")
    adapter.register_all()

    # 4. Agent 透明调用 MCP 工具
    agent = Agent(name="agent", llm_client=llm, tool_registry=registry)
    answer = await agent.arun("请使用 MCP 工具查询天气")
    print(answer)

    # 5. 断开
    await client.disconnect()

asyncio.run(main())
```

**支持的传输方式：**

| 传输方式 | 配置值 | 适用场景 |
|----------|---------|----------|
| stdio | `transport="stdio"` | 本地进程、npx 启动的服务器 |
| HTTP | `transport="http"` | 远程服务器、已部署的 MCP 服务 |
